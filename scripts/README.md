# Scripts

This directory contains all Jupyter notebooks and supporting files used for the class competition.

## Notebooks

### `eda.ipynb`
Exploratory data analysis of the training and test datasets. Includes class distribution analysis, text length analysis by class, and visualizations. Run this notebook first to understand the data before modeling.

### `[final_submission]model_lr.ipynb`
The primary classification notebook using TF-IDF features and Logistic Regression. Includes data loading, feature extraction, cross validation, hyperparameter tuning (testing values of C from 0.1 to 100), and final submission generation. This is the best performing model with a Kaggle macro F1 score of 0.9256.

### `model_nb.ipynb`
Classification notebook using Multinomial Naive Bayes with TF-IDF features. Included as a baseline comparison against Logistic Regression and SVM. Achieved a cross validation macro F1 of 0.8948.

### `model_svm.ipynb`
Classification notebook using a Linear Support Vector Machine (LinearSVC) with TF-IDF features. Achieved a cross validation macro F1 of 0.9200 and a Kaggle score of 0.9208.

## Supporting Files

### `eda_plots.png`
Visualization generated by `eda.ipynb` showing class distribution and text length by class. Used in the blog post.

### `launch-notebook`
Script for launching the Jupyter notebook server.

## Running Order

To reproduce results, run the notebooks in this order:

1. `eda.ipynb`
2. `[final_submission]model_lr.ipynb`
3. `model_nb.ipynb`
4. `model_svm.ipynb`

Submission files will be saved to the `submissions/` directory in the repo root.
